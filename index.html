AeroPulse: Flight Booking & Management System

AeroPulse is a full-stack flight booking and management platform built with Python, Django, and MySQL, designed for a smooth and modern user experience. It features advanced flight search, multi-step user registration, a personalized passenger dashboard, and a complete booking flow with e-tickets and invoices. A standout feature is the Global Price Map, an interactive visualization using Leaflet.js that shows worldwide travel affordability, turning flight discovery into a visual experience. Built on a scalable, 3NF-compliant database, AeroPulse is a strong foundation for real-world OTA systems.
